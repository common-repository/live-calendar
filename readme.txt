A event calendar like microsoft live calendar.

== Description ==

This is a event calendar plugin for wordpress, it works similar to microsoft live calensar .

You can make a calendar page or calendar sidebar or a upcoming event list.

[Demo](http://ziming.org/calendar)

== Installation ==

1. Unzip the archive and put the 'live-calendar' folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.

[Usage](http://ziming.org/dev/live-calendar)

= Usage =

1. Post a new page,  add "{CALENDAR} {TODAYS_EVENTS} {UPCOMING_EVENTS}" to page content.
2. Click the Manage Calendar submenu to create a new event, it's recommended to create a event category first.
3. In "Calendar Options" submenu to create you favarite setting.
4. Add "Event Calendar" widget to your sidebar widget.

[Advantage Usage](http://ziming.org/dev/live-calendar)

= Changing the CSS =

If you need to configure the CSS style of Live-Calendar, you can edit calendar.css file directly.
